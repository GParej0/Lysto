# ✅ Lysto - Task Management App

Lysto is a modern and responsive task management web application built with HTML, CSS, and JavaScript.  
This project was created to practice interactive frontend development, DOM manipulation, and dynamic task management features using vanilla JavaScript.

The app allows users to create, manage, and organize daily tasks through a clean and user-friendly interface.

## ✨ Features

- Add new tasks dynamically
- Mark tasks as completed
- Delete tasks
- Task persistence using Local Storage
- Saved tasks remain stored on the same device and browser
- Responsive design for mobile and desktop
- Clean and modern UI
- Interactive user experience
- Organized task layout
- Real-time DOM updates

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Local Storage API

## ⚙️ Installation

Clone the repository:

```bash

```

Open the project folder:

```bash
cd Lysto
```

Finally, open `index.html` in your browser.
